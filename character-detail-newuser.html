Character Profile

The Post-Migration Newcomer: This character represents a fresh user who wants to join DCP after the migration is complete, with no previous history on the Tezos blockchain or the DCP platform.

Test Case Objectives
  • Verify new user registration flow in the post-migration environment
  • Test wallet creation and connection processes for new accounts
  • Ensure Base L3 smart contract interaction works for first-time users
  • Validate onboarding experience on the migrated platform
  • Confirm token distribution mechanics function correctly for new users
Test Scenario

This test case is critical for ensuring that new user acquisition continues seamlessly after migration. While existing users need their data preserved, new users need a smooth onboarding experience in the new environment without any legacy considerations.

This character will test the complete workflow of discovering DCP, creating an account, connecting a wallet, receiving initial tokens (if applicable), and engaging with platform features - all without the context of migration.

Key Role: Testing the platform's ability to handle new users who have no prior relationship with the Tezos version of DCP, ensuring that post-migration growth isn't impeded by technical issues specific to new user onboarding.
